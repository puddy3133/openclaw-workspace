# 小红书账号配置指南

## 📱 需要提供的信息

为了让小乔帮你自动化运营小红书，需要以下信息：

### 1. 登录凭证（二选一）

**方案 A：Cookie 登录（推荐，更安全）**
```
- 小红书网页版的 Cookie 字符串
- 获取方式：登录小红书网页版 → F12 开发者工具 → Application/Storage → Cookies
```

**方案 B：账号密码**
```
- 小红书手机号/邮箱
- 登录密码
- 注意：此方式需要处理验证码
```

### 2. 发布配置

```json
{
  "xiaohongshu": {
    "account": {
      "type": "cookie",
      "cookie": "你的cookie字符串"
    },
    "preferences": {
      "default_topic_count": 5,
      "post_schedule": "immediate",
      "image_style": "magazine",
      "auto_publish": false
    }
  }
}
```

### 3. 内容偏好（可选）

- 你感兴趣的领域/话题
- 目标受众
- 发布频率偏好

---

## 🔐 安全提醒

1. **Cookie 方式更安全** - 不需要提供密码
2. **Cookie 会过期** - 通常 7-30 天需要更新
3. **建议设置 auto_publish: false** - 先生成内容，你审核后再发布
4. **账号信息存储** - 建议存储在环境变量或加密文件中

---

## 📤 如何提供给小乔

**方式 1：私聊发送（推荐）**
- 在当前对话中直接发送
- 小乔会存储到安全位置

**方式 2：文件方式**
- 创建一个 `.env` 文件
- 写入配置信息
- 小乔读取文件

**方式 3：分步提供**
- 先提供账号（不含密码）
- 小乔确认后，再提供密码/Cookie

---

## ⚠️ 风险提示

1. **自动化发布可能触发平台风控**
2. **建议先测试，再批量使用**
3. **不要频繁发布，模拟真人行为**
4. **小乔会记录发布日志，便于追踪**

---

准备好后，直接告诉小乔你的配置信息即可！
