# 个人进化系统 (Personal Evolution System)

> 灵感来自 OpenClaw Evolution，适配人类使用

## 核心概念

Evolution = 持续学习 + 刻意练习 + 定期复盘 + 渐进改进

---

## 系统架构

### 1. 输入层 (Input)
**每天收集：**
- 📚 阅读（文章/书籍/论文）
- 🎧 播客/视频/课程
- 💬 对话交流
- 🧪 实践尝试

**工具：**
- 飞书剪存（网页收藏）
- 微信读书/Kindle
- 备忘录（随时记录想法）

---

### 2. 处理层 (Process)
**每日（5-10分钟）：**
- [ ] 记录今天学到的一个新概念
- [ ] 记录一个失败/教训
- [ ] 记录一个灵感/想法

**每周（30分钟）：**
- [ ] 回顾本周输入
- [ ] 整理笔记到知识库
- [ ] 确定下周学习主题

**每月（2小时）：**
- [ ] 深度复盘本月成长
- [ ] 更新长期目标
- [ ] 调整进化方向

---

### 3. 输出层 (Output)
**写作：**
- 技术博客/公众号
- 读书笔记
- 项目文档

**实践：**
-  side project
-  开源贡献
-  技能应用

**分享：**
-  内部分享
-  社群交流
-  指导他人

---

### 4. 反馈层 (Feedback)
**指标追踪：**
| 指标 | 追踪方式 |
|------|----------|
| 阅读量 | 微信读书/Readwise |
| 写作量 | 字数统计 |
| 项目数 | GitHub/Trello |
| 技能树 | 自定义清单 |

**定期复盘：**
- 每周日晚：周回顾
- 每月最后一日：月复盘
- 每季度：深度审视

---

## 你的进化主题

基于你的背景，建议关注：

### 🔧 技术深度
- AI/LLM 应用开发
- 系统架构设计
- 工程效率提升

### 🎯 产品思维
- 需求分析

- 数据驱动

### 🤝 协作能力
- 技术领导力
- 跨团队沟通
- 知识分享

## 启动清单

**今天就做：**
- [ ] 创建一个 "Evolution" 文件夹（本地或云端）
- [ ] 设置每日提醒（晚上9点记录当天收获）
- [ ] 选择第一个进化主题（建议：AI Agent 开发）

**本周完成：**
- [ ] 建立阅读清单（3-5本/篇文章）
- [ ] 设置笔记模板
- [ ] 完成第一次周回顾

**持续进行：**
- [ ] 每日记录
- [ ] 每周复盘
- [ ] 每月深度审视
