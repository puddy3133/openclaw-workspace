## 使用方式设计

### 方式一：对话式调用（推荐）
```
用户：生成语音
小乔：好的，请提供文本内容

用户：小乔：你好呀！大乔：你好～小乔：今天天气不错。

小乔：收到！正在生成...
[生成中...]
小乔：✅ 语音已生成，包含 3 个片段：
- 小乔（片段1）：你好呀！
- 大乔（片段1）：你好～
- 小乔（片段2）：今天天气不错。
[播放按钮] [下载链接]
```

### 方式二：文件输入
```
用户：生成语音 /path/to/script.txt

小乔：读取文件成功，发现 5 句对话，角色：小乔(3句)、大乔(2句)
[生成中...]
```

### 方式三：直接命令
```
/tts 小乔：你好呀！
```

## 文本解析规则

### 支持格式

**格式 A：显式角色标记（推荐）**
```
小乔：你好呀！
大乔：你好～
小乔：今天天气不错。
```

**格式 B：带场景描述**
```
【场景：公园】
小乔：（微笑）你看那边的花多美！
大乔：（点头）是啊，春天来了。
```

**格式 C：纯文本（自动分配）**
```
你好呀！
你好～
```
→ 设置默认角色，或交替分配
