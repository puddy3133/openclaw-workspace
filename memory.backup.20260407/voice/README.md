## 快速使用指南

### 1. 手动生成（Hugging Face）
```
https://huggingface.co/spaces/mrfakename/E2-F5-TTS
```
- 上传对应角色的音频样本
- 输入参考文本和生成文本
- 点击 Submit 生成

### 2. 自动解析格式
```
小乔：你好呀，今天天气不错！
大乔：是啊，适合出去走走。
小乔：那我们去公园吧？
```
