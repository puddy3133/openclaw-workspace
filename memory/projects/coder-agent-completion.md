## 使用方法

### 直接调用 Coder
```javascript
sessions_spawn({
  label: "coder-task",
  runtime: "subagent", 
  mode: "run",
  task: "具体任务描述"
})
```

### 任务描述规范
- **目标**: 要实现什么功能
- **技术栈**: 使用什么语言/框架
- **约束**: 特殊要求或限制
- **输出**: 期望的产出物

### 输出格式
Coder 完成任务后会返回：
```
**状态**: 完成/失败
**概览**: [简要描述]
**进度**: 100%
**耗时**: XX 分钟
**产出**:
- 文件1: [路径]
- 文件2: [路径]
```

## 明早可直接使用的任务示例

**示例 1: 编写工具函数**
```
写一个 Python 函数，接收一个 URL，返回该网页的标题和 meta description。
要求：
- 使用 requests 和 BeautifulSoup
- 处理超时和异常
- 返回字典格式 {"title": "...", "description": "..."}
```

**示例 2: 创建 Web 组件**
```
创建一个 React 组件，实现一个带搜索功能的下拉选择器。
要求：
- 使用 TypeScript
- 支持键盘导航
- 样式使用 Tailwind CSS
```

**示例 3: 修复 Bug**
```
修复以下代码中的问题：[粘贴代码]
问题描述：[描述现象]
期望行为：[描述正确行为]
```
