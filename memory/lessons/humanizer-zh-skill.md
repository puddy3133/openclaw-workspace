# Humanizer-zh Skill 学习笔记

## 概述
**Humanizer-zh** 是一个 AI 写作去痕工具，用于识别并去除文本中的 AI 生成痕迹，使内容听起来更自然、更像人类书写。

**来源**: https://github.com/op7418/Humanizer-zh
**翻译自**: blader/humanizer
**参考**: hardikpandya/stop-slop

---

## 核心原则（5条）

1. **删除填充短语** - 去除开场白和强调性拐杖词
2. **打破公式结构** - 避免二元对比、戏剧性分段、修辞性设置
3. **变化节奏** - 混合句子长度。两项优于三项。段落结尾要多样化
4. **信任读者** - 直接陈述事实，跳过软化、辩解和手把手引导
5. **删除金句** - 如果听起来像可引用的语句，重写它

## 24种 AI 写作痕迹（四大类）

### 📝 内容模式（6种）
1. **过度强调意义、遗产和更广泛的趋势**
   - 警示词：作为/充当、标志着、见证了、是……的体现/证明、极其重要的/至关重要的、关键性的作用、反映了更广泛的、象征着、不断演变的格局
   - 改写：删除夸大的象征意义，直接陈述事实

2. **过度强调知名度和媒体报道**
   - 警示词：独立报道、地方/国家媒体、由知名专家撰写、活跃的社交媒体账号
   - 改写：提供具体来源和上下文

3. **以 -ing 结尾的肤浅分析**
   - 警示词：突出/强调……、确保……、反映/象征……、为……做出贡献、培养……
   - 改写：删除现在分词短语，直接陈述

4. **宣传和广告式语言**
   - 警示词：拥有（夸张用法）、充满活力的、丰富的、深刻的、增强其、展示、体现、致力于、自然之美、坐落于、令人叹为观止的
   - 改写：使用中性的描述性语言

5. **模糊归因和含糊措辞**
   - 警示词：行业报告显示、观察者指出、专家认为、一些批评者认为
   - 改写：提供具体的来源和引用

6. **提纲式的"挑战与未来展望"部分**
   - 警示词：尽管其……面临若干挑战……、尽管存在这些挑战、未来展望
   - 改写：删除公式化的挑战部分，直接陈述具体事实

### 🔤 语言和语法模式（6种）
7. **过度使用的"AI 词汇"**
   - 高频词：此外、至关重要、深入探讨、强调、持久的、增强、培养、获得、突出、相互作用、复杂/复杂性、关键、格局、展示、证明、宝贵的、充满活力的
   - 改写：使用更简单的同义词或直接删除

8. **避免使用"是"（系动词回避）**
   - 警示词：作为/代表/标志着/充当 [一个]、拥有/设有/提供 [一个]
   - 改写：使用简单的"是""有"

9. **否定式排比**
   - 结构："不仅……而且……""这不仅仅是关于……，而是……"
   - 改写：简化为直接陈述

10. **三段式法则过度使用**
    - 问题：LLM 强行将想法分成三组
    - 改写：改为两项或四项

11. **刻意换词（同义词循环）**
    - 问题：AI 的重复惩罚导致过度使用同义词
    - 改写：保持用词一致性

12. **虚假范围**
    - 结构："从 X 到 Y"但 X 和 Y 不在有意义的尺度上
    - 改写：直接列出具体内容

### 🎨 风格模式（6种）
13. **破折号过度使用** - 减少破折号使用
14. **粗体过度使用** - 删除不必要的粗体强调
15. **内联标题垂直列表** - 将粗体标题+冒号的列表改为流畅段落
16. **标题中的标题大写** - 中文不适用
17. **表情符号** - 删除装饰性表情符号
18. **弯引号** - 中文通常使用中文引号

### 💬 交流模式和填充词（6种）
19. **协作交流痕迹**
    - 警示词：希望这对您有帮助、当然！、您说得完全正确！、请告诉我
    - 改写：删除聊天机器人语气

20. **知识截止日期免责声明**
    - 警示词：截至 [日期]、根据我最后的训练更新、虽然具体细节有限
    - 改写：直接陈述已知事实

21. **谄媚/卑躬屈膝的语气** - 删除过于讨好的语言

22. **填充短语**
    - "为了实现这一目标" → "为了实现这一点"
    - "由于下雨的事实" → "因为下雨"
    - "在这个时间点" → "现在"
    - "系统具有处理的能力" → "系统可以处理"
    - "值得注意的是数据显示" → "数据显示"

23. **过度限定** - 删除多余的限定词（潜在地、可能、一些）

24. **通用积极结论** - 删除模糊的乐观结尾，改为具体事实

## 如何增加"人味"

- **有观点** - 不要只报告事实，要对它们做出反应
- **变化节奏** - 短促有力的句子 + 需要展开的长句
- **承认复杂性** - 真实的人有复杂感受
- **适当使用"我"** - 第一人称是诚实的表现
- **允许一些混乱** - 完美的结构感觉像算法
- **对感受要具体** - 用具体细节替代抽象概括

## 质量评分标准（总分50）

| 维度 | 评估标准 | 满分 |
|------|---------|------|
| 直接性 | 直截了当 vs 充满铺垫 | /10 |
| 节奏 | 长短交错 vs 机械重复 | /10 |
| 信任度 | 简洁明了 vs 过度解释 | /10 |
| 真实性 | 自然流畅 vs 机械生硬 | /10 |
| 精炼度 | 无冗余 vs 大量废话 | /10 |

**标准**: 45-50分优秀，35-44分良好，低于35分需重新修订

## 处理流程

1. 仔细阅读输入文本
2. 识别上述所有模式的实例
3. 重写每个有问题的部分
4. 确保修订后的文本：
   - 大声朗读时听起来自然
   - 自然地改变句子结构
   - 使用具体细节而不是模糊的主张
   - 为上下文保持适当的语气
   - 适当时使用简单的结构（是/有）
5. 呈现人性化版本 + 所做更改的简要总结
