## 场景
收到政府采购招标文件，需要分析技术标要求并制定响应策略。

## 方法
1. 读取招标文件全文
2. 提取 ★（实质性条款）/ ▲（重要参数）/ 评分权重
3. 识别演示要求和现场答辩要点
4. 输出结构化响应策略 + 风险清单

## 效果
快速识别招标文件中的关键得分点和风险项，为技术方案编写提供明确方向。

## 注意事项
- 必须区分实质性条款、重要参数、普通参数的风险等级
- ★ 实质性条款不满足 = 直接废标，最优先处理
