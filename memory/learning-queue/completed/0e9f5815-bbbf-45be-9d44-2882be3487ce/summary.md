## 核心内容

### 6个Prompt心法

| 心法 | 核心思想 | Prompt模板 |
|------|---------|-----------|
| **1. 让AI选定角色** | AI自己选最适合的专家角色 | "先别回答，请你先选一位最适合的领域顶尖名人专家来思考它" |
| **2. 苏格拉底式追问** | AI先追问再回答 | "请你在回答前，先问我问题。一次只问一个，直到你有95%的信心理解我的真实需求" |
| **3. 与AI辩论** | 让AI扮演反对者攻击你的观点 | "扮演反对者角色，从不同角度攻击我的想法，不用客气，直接指出漏洞" |
| **4. 预演失败** | 假设项目失败，提前复盘 | "假设这个项目失败了，什么时间点开始出现衰退信号？最致命的决策错误是什么？" |
| **5. 反向提示** | 给成品让AI倒推Prompt | "这是我想要的成品范例，请你倒推一个提示词，让我用它能稳定生成同风格的内容" |
| **6. 双层解释法** | 初学者版本+专业版本对照 | "请用两种方式回答：1. 初学者版本（洗脚城大爷也能听懂）2. 深度专业版本" |

### 关键洞察

1. **角色选择**: 让AI自己选专家，比用户指定更精准（如活动策划选Priya Parker）
2. **95%置信门槛**: 苏格拉底式追问的停止条件，既保证质量又避免无尽循环
3. **反对者角色**: 打破AI谄媚效应，获得客观判断
4. **预演失败**: 像影视Previz一样，正式开拍前踩坑
5. **反向工程**: 看到好内容时，让AI拆解学习结构和节奏
6. **双层解释**: 初学者版本用"洗脚城大爷"画像，更生活化不幼稚

### 心法本质
> "让对话变成你跟AI的协作，让提问变成你对AI的设计。"

## 提取的经验

1. **角色选择策略**: 不确定选什么角色时，让AI自己选顶尖专家
2. **追问停止条件**: 95%置信度，平衡质量与效率
3. **打破谄媚**: 主动要求AI扮演反对者，挑战自己的观点
4. **风险预演**: 项目开始前先做失败复盘，识别认知盲区
5. **反向学习**: 看到好内容时，让AI倒推Prompt来学习结构
6. **双层解释**: 初学者版用生活化类比，专业版保证准确性
