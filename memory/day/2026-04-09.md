## 重要说明

### ⚠️ Edge-TTS 不支持音色克隆

当前部署的 Edge-TTS 服务：
- ✓ 可以生成高质量的中文语音
- ✓ 有多种预设音色可选 (zh-CN-XiaoxiaoNeural 等)
- ✗ **不支持自定义音色克隆**
- ✗ **无法使用 sample2.mp3 作为参考音色**

### 如需音色克隆

需要以下任一方案：

1. **升级 Python 到 3.10+**
   - 安装 Homebrew Python
   - 重新部署 F5-TTS 或 GPT-SoVITS

2. **使用 CosyVoice**
   - 阿里开源，支持零样本克隆
   - 需要 Python 3.10+

3. **使用在线服务**
   - ElevenLabs API
   - 其他商业 TTS 服务
